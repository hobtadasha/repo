## Загальне

Цей репозиторій представляє собою вихідний код для проекту музикального програвача для ПК.

Програвач дозволяє не лише слухати збережені треки і створювати плейлісти, але також ділитися ними з друзями за допомогою посилань, без потреби в відправці файлів.

## Функції

- Програвання треків з диску або онлайн-ресурсу
- Створення та обмін плейлістами
- Вбудований еквалайзер
- Компактний дизайн
- Вбудована медіатека

## Запуск

Запустити програму в `bin/Player.exe` подвійним натисканням кнопки миші.
